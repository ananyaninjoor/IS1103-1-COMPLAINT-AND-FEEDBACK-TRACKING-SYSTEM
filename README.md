# Complaint and Feedback Tracking System

##  Project Overview

The Complaint and Feedback Tracking System is a web-based application designed to manage complaints and feedback in an organized and transparent manner.

In many organizations, complaints are handled manually through emails, phone calls, or written registers. This makes tracking difficult and reduces accountability. This system provides a structured digital solution where complaints can be submitted, tracked, and resolved efficiently.


##  Objectives

- Provide online complaint submission
- Generate unique complaint IDs
- Enable real-time status tracking
- Allow administrators to manage complaints
- Generate performance reports
- Improve transparency and accountability

##  SDLC Model Used

This project analyzes three SDLC models:

- Waterfall Model
- Incremental Model
- Spiral Model

After comparison, the **Incremental Model** was selected as the most suitable due to:
- Flexibility
- Support for changes
- Early working system delivery
- Better feedback incorporation
